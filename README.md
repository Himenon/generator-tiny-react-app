## Usage

```sh
npm i -g yo @himenon/generator-tiny-react-app

# yarn
yarn global add yo @himenon/generator-tiny-react-app
```

```sh
yo @himenon/tiny-react-app
```

## Development

```sh
git clone https://github.com/Himenon/generator-tiny-react-app.git
yarn
yarn run build
yarn link
```

Check

```
yo @himenon/tiny-react-app
```
